# PUBG Auto Invite Script

Автоматизация отправки приглашений в игре **PUBG** из списка ников.

Скрипт позволяет:

* Автоматически вставлять ники из текстового файла (`nicks.txt`)
* Искать игрока в игре и отправлять приглашение
* Работать на любом разрешении экрана (координаты сохраняются как проценты относительно окна)
* Быстро перенастраивать кнопки и поле ввода через горячие клавиши

---

## Установка

1. Скачайте файл `pubg_auto_clan_invite.exe`
3. Создайте файл `nicks.txt` рядом со скриптом и добавьте туда ники игроков, по одному в строке:

```
Player1
Player2
Player3
```

---

## Настройка

1. Откройте PUBG и наведите курсор на **кнопку Invite** → нажмите `F8`
2. Наведите курсор на **поле поиска игроков** → нажмите `F9`

Теперь скрипт запомнил позиции кнопки и поля ввода в процентных координатах окна игры.

---

## Использование

* **F6** — начать отправку приглашений
* **F7** — остановить скрипт

Скрипт будет:

1. Кликать в поле поиска
2. Очищать его
3. Вставлять ник из списка
4. Нажимать Enter (поиск)
5. Нажимать кнопку Invite

---

## Примечания

* Скрипт работает на **любом разрешении**, так как координаты сохраняются относительно окна PUBG
* Рекомендуется наводить курсор **по центру кнопки и поля**, чтобы клик срабатывал корректно
* Для больших списков ников можно добавлять **случайные задержки** для более «человеческого» поведения
